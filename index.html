BookAPI - Documentation

Overview

This api is connected to a database with all the books in the world. Just follow the documentation and try it out

Base URL

http://localhost:3000/book

Endpoints

GET /book

Retrieves all the books in the database

Response:

{ "title": "1984", "author": "George Orwell" }

GET /book/id

Retrieves one book by id from the database

Parameters:

int id

Response:

{ "title": "1984", "author": "George Orwell" }

POST /book

Insert a new book in the database

Parameters:

Body:

string title, string author

Response:

{ "title": "1984", "author": "George Orwell" }

PUT /book/id

Updates a book in the database by Id

Parameters:

int Id

Body:

string title, string author

Response:

{ "title": "1984", "author": "George Orwell" }

DELETE /book/id

Deletes a book by id in the database

Parameters:

int Id

Response:

{ "title": "1984", "author": "George Orwell" }
